# Overview

The sales dataset contains transactional tables capturing all customer purchases. It includes [orders](/tables/orders.md), [customers](/tables/customers.md), and [products](/tables/products.md).

# Tables

| Table | Description |
|-------|-------------|
| [orders](/tables/orders.md) | One row per completed customer order |
| [customers](/tables/customers.md) | Customer account records |
| [products](/tables/products.md) | Product catalog with pricing |
